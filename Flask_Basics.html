FlaskPython Web入門

一、Flask 是什麼

Flask 是 Python 的輕量級 Web Framework，適合初學者快速建立網站與 API。

二、安裝 Flask

python3 -m venv venv
source venv/bin/activate
pip install flask

三、第一個 Flask 網站

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return 'Hello Flask'

app.run(host='0.0.0.0', port=5000)

四、模板範例

from flask import render_template

@app.route('/hello')
def hello():
    return render_template('index.html')
