Raspberry Pi 常見課堂專題 20 選

依據目前課程材料整理，包含 Raspberry Pi 5、Python、GPIO、MQTT、Node-RED、Web、資料庫，以及 1080p Web CAM。

以下專題綜合了樹莓派常見教學方向與網路上高頻出現的專題類型，整理成適合課堂使用的版本。

材料與課程能力對應

  • Pi 5：主機平台、Linux、Python、Web、Docker
  • 1080p Web CAM：影像擷取、串流、監控、影像辨識
  • GPIO：LED、按鈕、蜂鳴器、感測器整合
  • MQTT / Node-RED：IoT 訊息傳輸與視覺化
  • MariaDB / Flask / Django：後台網站與資料儲存
我有刻意挑選「能在課堂中完成原型」且「能延伸成專題」的題目，而不是只列很炫但不適合教學的案子。

20 個常見且適合課堂的 Raspberry Pi 專題

1. 網頁版即時 Webcam 監看系統

Web CAMNginxPython

讓 Pi 5 接上 1080p Webcam，透過網頁顯示即時影像，是非常經典的樹莓派入門專題。

2. 動態偵測監視器

CameraMotion Detection

利用攝影機與影像差異檢測，當畫面有變化時拍照或錄影。

3. 智慧門鈴或訪客拍照系統

Web CAMGPIO

按鈕觸發攝影機拍照，再把照片存檔或上傳到網頁後台。

4. MQTT 攝影機事件發布系統

MQTTCamera

影像偵測或按鈕觸發後，透過 MQTT 發送事件通知，適合和 Node-RED 整合。

5. 人臉偵測或物件辨識入門系統

OpenCVAI

以 Webcam 做即時辨識，是結合 AI 與樹莓派的熱門專題方向。

6. 溫濕度監控系統

SensorsPython

讀取 DHT 感測器資料，顯示於終端機、網站或儀表板。

7. 光線自動照明系統

GPIOLight Sensor

光線不足時自動亮 LED 或控制繼電器。

8. PIR 人體移動偵測警報器

PIRBuzzer

感測有人移動時觸發蜂鳴器或傳送警報訊息。

9. 智慧教室出入偵測系統

PIRMQTT

用人體感測器偵測教室是否有人，進一步搭配儀表板顯示狀態。

10. GPIO 遠端控制平台

FlaskGPIO

用 Web 頁面控制 LED、蜂鳴器或其他輸出裝置。

11. Node-RED 智慧家居控制面板

Node-REDDashboard

透過儀表板控制 GPIO 與查看感測資料，非常適合課堂展示。

12. MQTT 感測器資料收集站

MQTTPython

由多個裝置送資料到 Pi，再統一儲存或顯示。

13. IoT 資料儀表板

Node-REDMariaDB

將感測資料存入資料庫，再以儀表板顯示歷史變化。

14. 環境監測網站

DjangoDatabase

以 Django 建立後台網站，管理感測器資料與紀錄。

15. Flask API 感測資料服務

FlaskAPI

將 GPIO 或感測資料封裝成 REST API，供前端或其他系統使用。

16. 樹莓派迷你 Web Server

NginxLinux

部署靜態網站或教學首頁，是 Web 與 Linux 的基礎專題。

17. 智慧相簿 / 縮時攝影系統

CameraStorage

定時拍照，產生影像紀錄或縮時影片。

18. 線上點名或影像打卡原型

CameraWeb

結合攝影機、Web 後台與資料儲存，適合作為進階專題發想。

19. 多介面感測器匯流排展示平台

I2CSPIUART

將不同介面的感測器接到 Pi 上，展示通訊差異與驅動方式。

20. 本地 AI 影像監測雛形

OllamaCameraAI

用樹莓派作為邊緣運算節點，結合影像資料與本地模型，做智慧監測雛形。

依課堂難度分級建議

  • 初階： LED 控制、按鈕、Web Server、溫濕度監控、MQTT 收發
  • 中階： Node-RED 儀表板、Flask GPIO 控制、MariaDB 感測資料儲存、PIR 警報
  • 進階： Django 後台、攝影機串流、動態偵測、AI 影像分析、本地部署整合

我最推薦的課堂專題方向

如果依你現在已有的教材與材料，我最推薦以下 5 題，因為完成度高、教學連貫、學生成就感也強：

  1. Node-RED + MQTT 感測器儀表板
  2. Web CAM 即時串流與動態偵測
  3. Flask / Django GPIO 控制平台
  4. MariaDB 感測資料紀錄網站
  5. GPIO + 感測器 + 網頁後台的整合型 IoT 專題
簡單說，你現在的器材組合很適合做「從硬體到網頁」的一條龍教學，這比單做某一顆感測器更有課程價值。
