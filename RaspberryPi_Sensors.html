Raspberry Pi 感測器整合教材

介紹 Raspberry Pi 與常見感測器模組整合方式，包含溫濕度、光敏、超音波、PIR 人體感測等。

適合 IoT、感測器介接、智慧裝置與 Python 控制課程。

Raspberry Pi感測器PythonIoT

一、教學目標

  • 理解感測器與 Raspberry Pi 的整合概念
  • 學會基本接線與安全注意事項
  • 能使用 Python 讀取常見感測器資料
  • 理解如何將感測資料應用於 IoT 系統

二、常見感測器

感測器用途介面
DHT11 / DHT22溫濕度數位
LDR 光敏電阻光線強弱類比模組或數位模組
HC-SR04距離量測Trig / Echo
PIR人體移動偵測數位
MQ 系列氣體偵測模組化輸出

三、DHT11 / DHT22 範例

import Adafruit_DHT
sensor = Adafruit_DHT.DHT11
pin = 4
humidity, temperature = Adafruit_DHT.read_retry(sensor, pin)
print("Temp:", temperature)
print("Humidity:", humidity)
不同系統環境的 DHT 套件可能不同，教學前建議先確認實機可用版本。

四、PIR 人體感測範例

from gpiozero import MotionSensor
from signal import pause
pir = MotionSensor(17)
pir.when_motion = lambda: print("偵測到人體移動")
pir.when_no_motion = lambda: print("沒有移動")
pause()

五、超音波感測器概念

HC-SR04 常用來量測距離，但 Echo 端為 5V，不能直接接 Raspberry Pi GPIO，需分壓或使用保護電路。

這是上課一定要強調的點，不然很容易燒板。

六、感測資料應用

  • 顯示在終端機
  • 上傳到 MQTT Broker
  • 顯示在 Node-RED 儀表板
  • 寫入資料庫
  • 透過 Flask / Django 顯示成網站

七、教學建議

  1. 先從 GPIO 入門
  2. 選一個感測器實作
  3. 讓學生把資料印出來
  4. 再延伸到 MQTT 或 Web 顯示
